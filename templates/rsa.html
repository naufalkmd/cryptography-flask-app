How RSA Works

RSA (Rivest‚ÄìShamir‚ÄìAdleman) is a public-key cryptosystem that is widely used for secure data transmission. Unlike the other ciphers we've seen, RSA uses two different keys:

  • A public key that anyone can use to encrypt messages
  • A private key that only the recipient knows, used for decryption

Key Generation Process:

  1. Choose two distinct prime numbers: p and q
  2. Calculate n = p √ó q
  3. Calculate œÜ(n) = (p-1) √ó (q-1)
  4. Choose e such that 1 < e < œÜ(n) and gcd(e, œÜ(n)) = 1
  5. Calculate d such that (d √ó e) mod œÜ(n) = 1
  6. Public key = (e, n), Private key = (d, n)

Encryption: c = me mod n

Decryption: m = cd mod n

Note: For educational purposes, we'll use small prime numbers. In real-world applications, RSA uses primes with hundreds of digits!

üìö RSA Example (Small Numbers)

Key Generation:

  1. Choose p = 17 and q = 19
  2. Calculate n = p √ó q = 17 √ó 19 = 323
  3. Calculate œÜ(n) = (p-1) √ó (q-1) = 16 √ó 18 = 288
  4. Choose e = 5 (since gcd(5, 288) = 1)
  5. Calculate d such that (d √ó 5) mod 288 = 1
  6. d = 173 (since 5 √ó 173 = 865 and 865 mod 288 = 1)
  7. Public key = (5, 323), Private key = (173, 323)

Encryption:

To encrypt the letter 'A' (which is 0 in our 0-25 system):

c = 05 mod 323 = 0

To encrypt the letter 'B' (which is 1 in our 0-25 system):

c = 15 mod 323 = 1

To encrypt the letter 'C' (which is 2 in our 0-25 system):

c = 25 mod 323 = 32

Decryption:

To decrypt 32 back to 'C':

m = 32173 mod 323 = 2 = 'C'
