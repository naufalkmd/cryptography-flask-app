Learning Cryptography Through Code

What is This App?

This is an educational web application built with Flask (Python) to help students learn about cryptographic algorithms in a hands-on, interactive way. It's designed specifically for Discrete Mathematics courses.

Educational Goals:

  • Understand how classic encryption algorithms work Kemed
  • See the mathematical foundations behind cryptography
  • Practice implementing algorithms in Python
  • Learn web development with Flask
  • Explore the evolution from simple to modern encryption

Technical Implementation

This application is built using:

  • Backend: Python with Flask framework
  • Frontend: HTML5, CSS3 (no JavaScript for simplicity)
  • Templating: Jinja2 (built into Flask)
  • Styling: Custom CSS with modern design principles

Why Flask? Flask is perfect for learning because it's simple, well-documented, and doesn't hide the underlying web concepts. You can see exactly how HTTP requests and responses work.
