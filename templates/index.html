What is Cryptography?

Cryptography is the practice of secure communication in the presence of third parties. It involves creating written or generated codes that allow information to be kept secret.

In this app, you'll learn about four important cryptographic algorithms:

  • Caesar Cipher: A simple substitution cipher where each letter is shifted by a fixed number of positions.
  • Vigen√®re Cipher: Uses a keyword to create multiple Caesar ciphers for different positions in the text.
  • Affine Cipher: Applies a mathematical function (ax + b) mod 26 to each letter position.
  • RSA: A modern public-key cryptosystem that uses prime numbers for secure communication.
